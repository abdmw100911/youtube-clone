YouTube Clone (HTML & CSS)

Description
This project is a static clone of the YouTube homepage built using only HTML and CSS. The goal of this project was to practice real-world frontend layout and styling by recreating the visual structure of YouTube.

No JavaScript, frameworks, or external libraries were used.

Features
The project includes a YouTube-style header with search bar and icons, a sidebar navigation menu, and a video grid layout showing thumbnails, channel images, titles, and video details. The layout is implemented using CSS Flexbox and Grid, with styles separated into multiple CSS files for better organization.

Technologies Used
HTML5
CSS3

Project Structure

channel images/
Contains channel profile images used in video cards

header icons/
Contains icons used in the top navigation bar

sidebar icons/
Contains icons used in the sidebar

thumbnails/
Contains video thumbnail images

styles/
general.css – common styles and resets
header.css – styles for the top header section
sidebar.css – styles for the sidebar navigation
video.css – styles for the video grid and cards

youtube.html
Main HTML file containing the page structure

How to Run
Clone or download the repository
Open youtube.html in any modern web browser

Purpose of the Project
This project was built to strengthen understanding of HTML structure, CSS layout techniques, file organization, and UI design before moving on to JavaScript-based frontend development.
